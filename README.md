# 3D Solar System Simulation

## Overview
A fully interactive 3D Solar System simulation developed using HTML, CSS, JavaScript, and Three.js. The project visualizes planetary motion, orbital mechanics, and real-time interactions in a modern 3D environment.

## Features
- Real-time 3D Solar System visualization
- Interactive planetary orbits
- Zoom and drag camera controls
- Planet information panel
- Hover tooltips
- Orbit animation controls
- Speed adjustment slider
- Saturn and Uranus ring effects
- Asteroid belt simulation
- Responsive futuristic UI design

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Three.js

## Functionalities
### Interactive Controls
- Drag to rotate camera
- Scroll to zoom
- Pause/Resume animation
- Toggle labels and orbit paths
- Reset camera view

### Planet Information
Each planet includes:
- Distance from Sun
- Diameter
- Mass
- Gravity
- Atmosphere
- Temperature
- Orbital period
- Fun facts
